## Пример 3
### Решение

Пусть $X$ - дискретная случайная величина с $E[X] = a$ и $Var[X] = b^2 \neq 0$. Мы строим новую случайную величину, которая является функцией от $X$:

$$
    Y = g(X) =  \frac{X - a}{b}
$$

Найдите $E[Y]$ и $Var[Y]$.

\textit{Решение.}
    \begin{gather*}
        E[Y] = E \left[ \frac{X - a}{b} \right] = E \left[ \frac{X}{b} + \frac{-a}{b}\right] =  \frac{1}{b}E[X] - \frac{a}{b} = \frac{a}{b} - \frac{a}{b} = 0\\
        Var[Y] = Var \left[ \frac{X - a}{b} \right] = Var \left[ \frac{X}{b} + \frac{-a}{b}\right] = \frac{1}{b^2}Var[X] = \frac{b^2}{b^2} = 1
    \end{gather*}

## Распределение Бернулли

Начнем с простейшего случая - **одного случайного эксперимента** с двумя возможными исходами.

:::{.callout-definition}
**Распределение Бернулли** - это распределение случайной величины $X$, которая принимает только два значения:
$$
X = \begin{cases}
1 & \text{с вероятностью } p \text{ ("успех")} \\
0 & \text{с вероятностью } q = 1-p \text{ ("неудача")}
\end{cases}
$$
Обозначается как $X \sim Bern(p)$, где $p \in [0, 1]$ - параметр распределения.
:::

## Характеристики распределения Бернулли

::: {.callout-tip title="Основные характеристики"}
Для случайной величины $X \sim Bern(p)$:
\begin{itemize}
    \item Математическое ожидание:
    $$E[X] = 0 \cdot (1-p) + 1 \cdot p = p$$

    \item Дисперсия:
    $$Var[X] = E[X^2] - (E[X])^2 = p - p^2 = p(1-p)$$

    \item Стандартное отклонение:
    $$\sigma = \sqrt{p(1-p)}$$
\end{itemize}
:::

## Примеры распределения Бернулли

\begin{enumerate}
    \item \textbf{Подбрасывание монеты}: "Орел" = 1, "Решка" = 0, $p = 0.5$ (для честной монеты)
    
    \item \textbf{Производственный контроль}: "Дефектное изделие" = 1, "Качественное" = 0
    
    \item \textbf{Медицинский тест}: "Положительный результат" = 1, "Отрицательный" = 0
    
    \item \textbf{Маркетинг}: "Покупка товара" = 1, "Отказ от покупки" = 0
\end{enumerate}

## От одного испытания к процессу

\begin{itemize}
    \item Что если мы хотим провести несколько независимых испытаний Бернулли?

    \item Например: подбросить монету 10 раз, проверить 100 изделий на дефекты, провести 50 медицинских тестов.

    \item Это приводит нас к понятию процесса Бернулли - последовательности независимых испытаний Бернулли.
\end{itemize}

## Процесс Бернулли

:::{.callout-definition}
**Процесс Бернулли** - это последовательность независимых одинаково распределенных случайных величин $X_1, X_2, \ldots, X_n$, где каждая $X_i \sim Bern(p)$:
$$
X_i = \begin{cases}
1 & \text{с вероятностью } p \text{ ("успех")} \\
0 & \text{с вероятностью } 1-p \text{ ("неудача")}
\end{cases}
$$
:::


Ключевое отличие от одного испытания: теперь у нас есть **последовательность** результатов.

### Свойства процесса Бернулли

\begin{enumerate}
    \item \textbf{Независимость}: Результат каждого испытания $X_i$ не зависит от результатов других испытаний
    
    \item \textbf{Два исхода}: Каждое испытание может закончиться только успехом (1) или неудачей (0)
    
    \item \textbf{Постоянная вероятность}: Вероятность успеха $p$ не изменяется от испытания к испытанию
\end{enumerate}

## Примеры процесса Бернулли

\begin{enumerate}
    \item \textbf{Серия подбрасываний монеты}: Последовательность $(X_1, X_2, \ldots, X_{10})$ для 10 подбрасываний
    
    \item \textbf{Контроль качества}: Проверка партии из $n$ изделий, где каждое может быть дефектным независимо от других
    
    \item \textbf{Клинические испытания}: Тестирование лекарства на $n$ пациентах, где каждый может показать положительную реакцию
\end{enumerate}

## Биномиальное распределение

- Теперь, когда у нас есть процесс Бернулли, возникает естественный вопрос: **"Сколько успехов мы получим в $n$ испытаниях?"**

:::{.callout-definition}
**Биномиальное распределение** описывает количество успехов в $n$ независимых испытаниях Бернулли с вероятностью успеха $p$.
Если $X_1, X_2, \ldots, X_n$ - процесс Бернулли с параметром $p$, то случайная величина:
$$Y = X_1 + X_2 + \ldots + X_n$$
имеет биномиальное распределение и обозначается $Y \sim Bin(n, p)$.
$Y$ может принимать значения $0, 1, 2, \ldots, n$ (от $0$ до $n$ успехов).
:::


## Функция вероятности биномиального распределения

Вероятность получить ровно $k$ успехов в $n$ испытаниях:

$$
    P(Y = k) = C_n^k p^k (1-p)^{n-k}
$$

::: {.callout-tip title="Интуиция формулы"}
\begin{itemize}
    \item $p^k$ - вероятность $k$ успехов
    \item $(1-p)^{n-k}$ - вероятность $(n-k)$ неудач  
    \item $C_n^k$ - количество способов выбрать $k$ позиций из $n$ для размещения успехов
\end{itemize}
:::

## Характеристики биномиального распределения

::: {.callout-tip title="Основные характеристики"}
Для случайной величины $Y \sim Bin(n, p)$:

**Математическое ожидание:**
$$E[Y] = np$$

**Дисперсия:**
$$Var[Y] = np(1-p)$$

**Стандартное отклонение:**
$$\sigma = \sqrt{np(1-p)}$$
:::

**Вывод математического ожидания:** $E[Y] = E[X_1 + X_2 + \ldots + X_n] = E[X_1] + E[X_2] + \ldots + E[X_n] = p + p + \ldots + p = np$

## Визуализация биномиального распределения

::: {.callout-note title="Наблюдения"}
**p < 0.5** (например, p = 0.3): Распределение смещено влево, большинство значений сосредоточено в области малых k

**p = 0.5**: Распределение симметрично относительно среднего значения

**p > 0.5** (например, p = 0.7): Распределение смещено вправо, большинство значений сосредоточено в области больших k
:::

## Процесс Бернулли, биномиальное распределение
### Пример 1: подбрасывание монеты

Подбросим честную монету 10 раз. Какова вероятность получить ровно 6 орлов?

- $n = 10$, $p = 0.5$, $k = 6$, $Y \sim Bin(10, 0.5)$

$$P(Y = 6) = C_{10}^6 (0.5)^6 (0.5)^4 = \frac{10!}{6! \cdot 4!} (0.5)^{10}$$

$$= \frac{10 \cdot 9 \cdot 8 \cdot 7}{4 \cdot 3 \cdot 2 \cdot 1} \cdot \frac{1}{1024} = 210 \cdot \frac{1}{1024} \approx 0.205$$

**Ожидаемое количество орлов:** $E[Y] = 10 \cdot 0.5 = 5$
